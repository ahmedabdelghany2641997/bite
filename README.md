### 🚀 Key Features
* **Real-time Tracking:** Integration with **Google Maps API** for live rider tracking.
* **Smart Cart Logic:** Advanced state management to handle complex orders and custom meal options.
* **Performance:** High-speed data fetching with **Dio** and custom interceptors.
* **Caching:** Offline support using **Hive** for a lag-free menu browsing experience.
* **Clean UI:** Pixel-perfect implementation based on modern Material 3 design.

### 🏗️ Software Architecture
Project follows **Clean Architecture** principles to ensure scalability and testability:
* **Domain Layer:** Entities, UseCases, and Repository Interfaces.
* **Data Layer:** Models, DataSources (Remote/Local), and Repository Implementations.
* **Presentation Layer:** BLoC/Cubit for state handling and reusable UI Widgets.

### 🧪 Engineering Excellence
- [x] **SOLID Principles** applied for maintainable code.
- [x] **Dependency Injection** using `GetIt`.
- [x] **Error Handling** with functional programming approach (Either/Dartz).
